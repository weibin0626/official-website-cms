## 部署建议

1. **生产环境** 使用 PostgreSQL 替换 SQLite
2. **JWT_SECRET** 必须修改为强随机值
3. **端口配置** 通过环境变量 `PORT` 指定，避免冲突
4. **上传目录** 确保 `uploads/` 有写入权限，建议挂载独立存储
5. **HTTPS** 生产环境必须启用 SSL
